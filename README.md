# SAS Project

This workspace is organized for SAS development with separate folders for source programs, macros, data, and outputs.

## Structure

- `programs/` - Main SAS programs
- `macros/` - Reusable macro definitions
- `data/` - Raw and derived data files
- `outputs/` - SAS-generated output files
- `results/` - Analysis results and reports
- `docs/` - Project documentation

## Getting Started

1. Put raw or sample datasets in `data/`.
2. Write analysis programs in `programs/`.
3. Use `macros/` for shared SAS macros.
4. Save generated outputs in `outputs/` and final results in `results/`.
